# To-Do List

## Описание проекта
To-Do List — это простое веб-приложение для работы со списком задач.  
Проект создан на JavaScript.

## Цель проекта

Цель проекта — показать практические навыки работы с JavaScript, DOM, событиями, массивами объектов, пользовательским вводом и модульной структурой кода.

## Основные функции

- добавление задач;
- удаление задач;
- редактирование задач;
- отметка задачи как выполненной;
- поиск задач;
- фильтрация задач по статусу;
- сортировка задач;
- валидация пользовательского ввода;
- динамическое создание HTML-элементов через JavaScript;
- адаптивный интерфейс.

## Используемые технологии

- HTML;
- CSS;
- JavaScript ES6+.

## Инструкция по запуску

1. Скачать папку проекта.
2. Открыть файл `index.html` в браузере.
3. Начать работу со списком задач.

## Зависимости

Проект не использует сторонние библиотеки и фреймворки.

## Пример использования

Чтобы добавить задачу, нужно ввести текст в поле ввода и нажать кнопку "Добавить".

Пример задачи:

```javascript
{
    id: 1,
    text: "Сделать домашнее задание",
    completed: false,
    createdAt: Date.now()
}
